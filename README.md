# Kotlin App – API Connector

Este proyecto fue desarrollado como parte de un examen técnico de la materia TC2007B, cuyo propósito es *conectar una aplicación hecha en Kotlin con una API externa*. La aplicación demuestra el consumo de endpoints, el manejo de respuestas HTTP y la organización adecuada del código.
